# Fonty — Vzpomínkář

Dva fonty z logo manuálu. **Obojí zdarma na Google Fonts** → pro web netřeba stahovat
(načítá se z CDN, viz `../README.md`). Odkazy ke stažení (kdybys chtěl soubory do počítače / do tisku):

## Primární (nadpisy) — Bree Serif
- Foundry: TypeTogether (Veronika Burian & José Scaglione)
- Ke stažení: https://fonts.google.com/specimen/Bree+Serif
- ⚠️ Free Google verze = jen řez Regular (400). Plná rodina **Bree** (víc řezů + vzpřímená
  kurzíva, popsaná v manuálu) je **placená** licence TypeTogether — to bys musel koupit zvlášť,
  pokud bys chtěl víc tučností v nadpisech.

## Sekundární (UI / text) — Host Grotesk
- Foundry: Element Type (postaveno na Poppins, Indian Type Foundry)
- Ke stažení: https://fonts.google.com/specimen/Host+Grotesk
- Variabilní, plný rozsah řezů (300–700+).

---

Až mi fonty stáhneš, hoď `.woff2` / `.ttf` sem do téhle složky a já je v případě potřeby
zapojím přes `@font-face` (místo Google CDN). Pro teď stačí CDN.
