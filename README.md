# Vzpomínkář — editorial mockup (v2)

Static single-page HTML mockup exploring the editorial-heritage direction for the Vzpomínkář homepage, inspired by [nenek.eu](https://www.nenek.eu).

## Stack

- Single `index.html` — no build, no JS framework
- **PP Pangaia** (Pangram Pangram) for display type — licensed, OTF served locally
- **Instrument Sans** (Google Fonts) for body
- **Caveat** (Google Fonts) for handwritten flourishes
- ~30 lines of vanilla JS for scroll-driven hero animation

## Local preview

```sh
open index.html
```

That's it.

## Deploy

Auto-deploys to Vercel on push.
